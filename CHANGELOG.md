# 更新日志

所有重要的项目更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.7.0] - 2024-02-02

### 新增
- ✨ UDP 广播自动发现功能 - 手机端自动探测局域网内的桌面端
- ✨ 服务器选择界面 - 显示所有发现的服务器供用户选择
- ✨ iOS 下载优化 - 使用 Fetch + Blob 避免页面跳转
- ✨ 实时下载进度显示 - 支持显示下载百分比和进度条
- ✨ 移动端导航悬浮优化 - 导航栏悬浮在内容区域底部

### 改进
- 🎨 优化移动端布局 - 内容区域有足够的底部空间
- 🎨 改进滚动体验 - 滚动条不占用内容宽度
- ⚡ 智能扫描策略 - 2-5秒快速发现服务器
- 📱 优化 PWA 体验 - 更好的移动端适配

### 修复
- 🐛 修复 iOS 下载后无法返回应用的问题
- 🐛 修复移动端导航遮挡内容的问题
- 🐛 修复设备发现偶尔失败的问题
- 🐛 修复下载进度显示不准确的问题

### 技术细节
- 实现 UDPDiscovery 类用于设备发现
- 添加 useUDPDiscovery Hook 管理发现状态
- 创建 ServerSelector 组件用于服务器选择
- iOS 设备检测和兼容的下载方式
- 底部导航绝对定位悬浮实现

### 文档
- 📖 添加 UDP 发现功能指南
- 📖 添加 iOS 下载优化说明
- 📖 创建完整的 README.md
- 📖 添加上线检查清单
