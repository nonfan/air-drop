# 移动端 UI 测试指南

## 快速测试步骤

### 1. 启动开发服务器
开发服务器已启动：
- **本地访问**: http://localhost:5174/
- **手机访问**: http://192.168.0.2:5174/

### 2. 桌面端测试（浏览器）
1. 打开 http://localhost:5174/
2. 验证显示：
   - ✅ 左侧侧边栏
   - ✅ 顶部标题栏
   - ✅ 底部版本信息
   - ❌ 不显示底部导航栏

### 3. 移动端测试（Chrome DevTools）

#### 打开设备模拟器
1. 按 `F12` 打开 Chrome DevTools
2. 按 `Ctrl+Shift+M` 切换到设备模拟模式
3. 选择设备：iPhone 12 Pro 或 iPhone 14 Pro

#### 测试清单

##### 顶部栏
- [ ] 显示 "你好, {设备名称} 👋"
- [ ] 右上角显示设置按钮（齿轮图标）
- [ ] 显示四个快捷按钮：
  - [ ] 文件（绿色，文件图标）
  - [ ] 照片（蓝色，图片图标）
  - [ ] 文本（粉色，文档图标）
  - [ ] 文件夹（紫色，日历图标）

##### 快捷按钮功能
- [ ] 点击"文件"按钮 → 打开文件选择器
- [ ] 点击"照片"按钮 → 打开文件选择器
- [ ] 点击"文本"按钮 → 弹出文本输入弹窗
- [ ] 点击"文件夹"按钮 → 跳转到历史记录页面

##### 文本输入弹窗
- [ ] 点击"文本"按钮后弹窗从底部滑入
- [ ] 弹窗显示：
  - [ ] 标题 "发送文本"
  - [ ] 关闭按钮（X 图标）
  - [ ] 文本输入框
  - [ ] 发送按钮
- [ ] 点击背景黑色遮罩 → 关闭弹窗
- [ ] 点击关闭按钮 → 关闭弹窗
- [ ] 输入文本后点击发送 → 发送并关闭弹窗
- [ ] 未选择设备时发送按钮禁用（灰色）

##### 底部导航栏
- [ ] 显示三个导航按钮：
  - [ ] 首页（房子图标）
  - [ ] 传输（文件图标）
  - [ ] 我的（用户图标）
- [ ] 当前页面的按钮高亮显示（蓝色）
- [ ] 点击导航按钮切换页面
- [ ] 底部导航栏不被遮挡

##### 隐藏元素
- [ ] 侧边栏隐藏
- [ ] 桌面端顶部标题栏隐藏
- [ ] 桌面端底部版本信息隐藏

## 常见问题

### Q: 底部导航栏被系统栏遮挡
A: 这是 iOS 安全区域问题，已添加 `safe-area-bottom` 类，在真机上应该正常。

### Q: 弹窗没有动画
A: 检查 CSS 文件是否包含 `animate-slide-up` 类和 `@keyframes slideUp` 动画。

### Q: 快捷按钮点击没反应
A: 检查浏览器控制台是否有错误，确保 Socket.IO 连接正常。

### Q: 设备列表为空
A: 需要先启动桌面端应用，或者连接到已运行的服务器。
