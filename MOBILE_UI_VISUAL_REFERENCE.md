## 🎨 颜色方案

### 快捷按钮
- **文件**：`bg-green-500/10` + `text-green-600`
- **照片**：`bg-blue-500/10` + `text-blue-600`
- **文本**：`bg-pink-500/10` + `text-pink-600`
- **文件夹**：`bg-purple-500/10` + `text-purple-600`

### 状态颜色
- **选中**：`text-accent` (#3b82f6 蓝色)
- **未选中**：`text-muted` (灰色)
- **悬停**：`hover:text-foreground`

### 背景颜色
- **主背景**：`bg-background` (#0f0f12)
- **次级背景**：`bg-secondary` (#18181c)
- **三级背景**：`bg-tertiary` (#1f1f24)
- **悬停背景**：`bg-hover` (#26262c)

## 📐 尺寸规范

### 间距
- 顶部栏内边距：`px-4 py-3`
- 快捷按钮间距：`gap-3`
- 底部导航内边距：`px-4 py-2`

### 圆角
- 快捷按钮：`rounded-xl` (12px)
- 弹窗顶部：`rounded-t-3xl` (24px)
- 一般卡片：`rounded-lg` (8px)

### 图标尺寸
- 快捷按钮图标：`w-6 h-6`
- 底部导航图标：`w-6 h-6`
- 设置按钮图标：`w-5 h-5`

### 按钮尺寸
- 快捷按钮圆形背景：`w-12 h-12`
- 设置按钮：`w-9 h-9`
- 弹窗关闭按钮：`w-8 h-8`

## 🎬 动画效果

### 弹窗滑入
```css
@keyframes slideUp {
  from {
    opacity: 0;
    transform: translateY(100%);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
- 持续时间：0.3s
- 缓动函数：cubic-bezier(0.4, 0, 0.2, 1)

### 按钮悬停
- 过渡：`transition-colors`
- 背景变化：`hover:bg-hover`

### 导航切换
- 颜色过渡：`transition-colors`
- 选中高亮：`text-accent`

## 📱 iOS 安全区域

### 底部导航栏
```css
.safe-area-bottom {
  padding-bottom: env(safe-area-inset-bottom);
}
```

### 效果
- iPhone X 及以上：自动添加底部安全区域
- 其他设备：无额外间距
- 确保导航栏不被系统栏遮挡

## 🔄 响应式断点

### Tailwind 断点
- `md:` → 768px
- 移动端：`< 768px`
- 桌面端：`≥ 768px`

### 显示/隐藏
- 移动端显示：`md:hidden`
- 桌面端显示：`hidden md:flex` 或 `hidden md:block`

## 💡 交互细节

### 触摸优化
```css
* {
  -webkit-tap-highlight-color: transparent;
  -webkit-touch-callout: none;
}
```

### 防止缩放
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

### 点击反馈
- 按钮：`hover:bg-hover`
- 导航：`hover:text-foreground`
- 快捷按钮：`hover:bg-{color}-500/20`

```
z-index 层级：

```

## 🎯 关键特性

1. **响应式设计**：自动适配桌面和移动端
2. **iOS 优化**：安全区域适配，触摸优化
3. **流畅动画**：弹窗滑入，按钮过渡
4. **直观导航**：底部导航栏，快捷按钮
5. **现代设计**：圆角卡片，半透明遮罩
