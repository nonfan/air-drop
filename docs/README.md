# Airdrop 官网

这是 Airdrop 的产品官网，支持 GitHub Pages 部署。

## 需要添加的图片

请将以下截图添加到 `docs/` 目录：

1. **preview-desktop.png** - 桌面端应用截图
   - 建议尺寸：1040x780 像素
   - 截取应用主界面

2. **preview-mobile.png** - 手机端网页截图
   - 建议尺寸：360x780 像素
   - 截取手机浏览器访问的界面

## 部署到 GitHub Pages

1. 进入仓库 Settings → Pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 `main`，文件夹选择 `/docs`
4. 点击 Save

部署完成后，访问 `https://nonfan.github.io/air-drop/` 即可看到官网。
