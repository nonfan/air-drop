特性

Airdrop 会自动检测同一 Wi-Fi 网络上的设备，无需手动输入 IP 地址，在后台轻松传输——就像一个跨平台的 AirDrop 一样。

手机扫码连接

手机扫描二维码即可通过浏览器访问，无需安装任何 App，支持 iOS 和 Android。

电脑互传

支持同一局域网下多台电脑之间的文件传输，自动发现设备，点击即可发送。

文件传输

支持拖拽或选择文件，支持多文件批量传输，传输速度快，稳定可靠。

文本传输

快速在设备间共享文本内容，点击即可复制，方便分享链接、代码等。

传输记录

完整的文件和文本传输历史记录，按时间排序，支持一键清空。

系统通知

收到文件时桌面通知提醒，不会错过任何传输，可在设置中开关。

常见问题

为什么手机扫码后无法访问？

请确保手机和电脑连接在同一个 Wi-Fi 网络下。如果仍无法访问，可能是电脑防火墙阻止了连接，请在防火墙设置中允许 Airdrop 访问网络。

传输速度取决于什么？

传输速度主要取决于你的局域网带宽和路由器性能。在千兆局域网下，传输速度可达 100MB/s 以上。

支持哪些操作系统？

目前桌面端支持 Windows 系统。手机端通过浏览器访问，支持 iOS Safari 和 Android Chrome 等主流浏览器。

文件大小有限制吗？

没有文件大小限制，但建议单个文件不超过 4GB，以获得最佳传输体验。
